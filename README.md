![QCalcFileHash images](http://dansoft.krasnokamensk.ru/data/1018/qcalcfilehash.png)


### How do I get set up? ###
qmake

lupdate ./QCalcFileHash.pro

lrelease ./QCalcFileHash.pro

mkdir /usr/share/qcalcfilehash/

cp *.qm /usr/share/qcalcfilehash/

make
